Description

back to top

A theme options module to use in conjunction with the JustBlack theme.

Note: this module is optional. The JustBlack Theme is fully functioning without this module or without any options set.

Requirements

back to top
This module requires webtrees version 1.4.2 or later.

Installing the module

back to top
Installation is very simple. Upload the folder 'justblack_theme_options' to your server's webtrees /modules_v3 folder. Then login to webtrees as administrator and go to Module Administration (Admin -> Module Administration). Look down the list of installed modules for JustBlack Theme Options" and ensure it is ticked as "enabled".

IMPORTANT: In order to retrieve the configuration options you must click again on the module link in the menu. This will refresh the menu and add the Fancy Tree View configuration page to the admin menu.

After the module is installed go to the module configuration page to set some options.

Module options

back to top
On the JustBlack Theme Options configuration page you can set the following options:
  • Set position and size of the Tree title;
  • Use the default header image, upload a custom one or opt for no header image.
  • It is possible to resize the newly uploaded header image to the default values of the JustBlack Theme (800x150px).
  • In case you choose a custom header or choose not to use a header image you can set the height of the header area (You can not use this option in conjunction with the resize option)..
  • Opt for flags as language menu or use the default dropdown menu. Advice is not to use this option if you have more then 10 languages installed; Remove the languages you won't need from the language folder of your webtrees installation.
  • Use a compact 'View' menu. In the compact version of the topmenu the menu-items Charts, Lists, Calendar and Reports (optional) will be put together in one mainmenu-item.
  • Order the main menu items. Ordering menu items here is more flexible then the sort option webtrees offers by default. Here you can shuffle ALL main menu items (except the homepage and my page menu).

Custom translations

back to top

The JustBlack Theme Options Modules only uses language strings in the administration area. Where possible this module uses existing webtrees language items. These will therefore translate automatically wherever they are used. We have included new language items in the same "WT_I18N::translate()" format. This means you can translate these into any other language using webtrees "custom translation" feature, described here: Wiki: Custom translation.

If you are working with po files you can use the English po file added in the module language folder, to start translate the module in other language(s). The Dutch translation file is already added to the module language folder. In stead of adding your custom translation file(s) to the folder data/language, you may opt to keep them seperate from other translated files by adding them to the folder modules_v3/fancy_treeview/language. Csv of php files as decribed on the Wiki could be used as well.
