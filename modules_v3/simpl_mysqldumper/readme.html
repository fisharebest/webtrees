DESCRIPTION

back to top

This is a very small, simple module with a very powerful purpose. It provides an ability to include the excellent "MySQLDumper" backup software into your webtrees administration panel options. Throughout webtrees documentation you are very sensibly advised to back up your database at frequent intervals, and especially before any upgrade. If your webhost provides a tool like PhpMyAdmin or cPanel you should, if you know how, be able to use that. But for greater flexibility, simplicty and convenience MySQLDumper is hard to beat.
See side panel for MySQLDumper's own description of its features

I find the abilty to automatically email you a zipped copy of the complete backup particularly helpful. To go further, perhasp with some help from you web host, you set up a "cron job" to make the entire process automated, emailing you a fresh backup every few days or weeks.

Once installed you will see a new link in the module-specific section of the administration panel, bottom left corner, similar to that shown here to the left.

REQUIREMENTS

back to top
This version requires webtrees version 1.4.0 or later.

You also need a working installation of MySQLDumper. This is NOT provided as part of this module. See Installation below.

INSTALLATION

back to top
  1. Before doing anything with these files, download, install the latest version of MySQLDumper from www.mysqldumper.net. Where you install it is not ctitical as this module simply requires you to provide a working link to the installation. However, I recommend, for simplicty and security that you install it in the same part of your web server as your webtrees files, perhaps in a parallel directory. After installation, access MySQLDumper directly to set it up and to check it is fully functional. There is no point wasting time with this module until you have a working installation of MySQLDumper. [ NOTE: If you have any problems installing or using MySQLDumper please contact them at their website for support. They provide excellent service. www.mysqldumper.net]
  2. Once MySQLDumper is working you can proceed with installing this webtrees module. Unzip the files and upload them to the webtrees/modules_vX/ folder on your web server so that you have a new folder there called "simpl_mysqldumper". (Note: "X" = "3" at the time of writing this, but may change in future releases)
  3. Login to webtrees as administrator, go to AdministrationModules, and find the module. It will be called "Simple_SQL Backup". Check it has a tick for "Enabled"
  4. Finally click SAVE, to complete the installation. Refresh your administration screen ("F5" in most browsers) and you will see the link in the menu, although it will not work yet. See Setup below.

SETUP

back to top
There is just one item requiring setup before this module can be used. Open the file, from this module, called admin_mysqldumper.php. At line 36 replace the entry "mysqldumper/index.php" with the correct path to the index.php file in your own MySQLDumper installation. Then save the revised file back to your web server.

UPGRADING

back to top
Upgrading is extremely simple. Just copy these new files over the top of your existing ones, or if you prefer delete the old files and replace with these new ones. Do not make any changes to your separate MySqlDumper folder or files. Just take care if you modified line 36 as described in SETUP above. It will need the same change in the new file.
