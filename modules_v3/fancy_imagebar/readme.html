Requirements

back to top
This module requires webtrees version 1.4.0 or later.

Installing the module

back to top
Installation is very simple. Extract the files from their zip container into your server's webtrees /modules_v3 folder; this will create a new folder in /modules_v3 called "fancy_imagebar" containing all files. Then login to webtrees as administrator and go to Module Administration (Admin -> Module Administration). Look down the list of installed modules for Fancy Image Bar" and ensure it is ticked as "enabled".

IMPORTANT: In order to retrieve the configuration options you must click again on the module link in the menu. This will refresh the menu and add the Fancy Image Bar configuration page to the admin menu.

After the module is installed go to the module configuration page to choose the images you want to show in the imagebar for each tree you have installed. Here you can also set some other options.
Once you have saved the configuration, the Fancy Imagebar will be shown on the frontsite.

Module options

back to top
On the Fancy Image Bar configuration page for each installed tree you can choose which images you want to show in the Fancy Imagebar and on which page (homepage or my page or both).

You can also set the following options:

  • Show the imagebar on the Homepage and/or My Page
  • Choose between Sepia (and the amount of sepia), Black and White or original colors;
  • Set the size (height) of the imagebar.
  • Show random images or not.

FAQ

back to top

Here are a few questions / answers that might help....

The administration page is in English. Why?
Where possible the Fancy Imagebar module uses existing webtrees language items. These will therefore translate automatically wherever they are used. We have included new language items in the same "WT_I18N::translate()" format. This means you can translate these into any other language using webtrees "custom translation" feature, descibed here: Wiki: Custom translation.

Will the Fancy Imagebar look fine in any theme?
The Fancy Imagebar Module comes with a stylesheet for each default webtrees theme and the JustBlack theme. The Fancy Imagebar looks fine in any of those themes.

Is the Fancy Imagebar compatible with all browsers?
Not likely to work well with any version of Internet Explorer older than IE9; but all modern browsers should be OK.
