Description

back to top
A sidebar module that provides quick links to popular research web sites, using the individuals name as the search reference.

Requirements

back to top
This module requires webtrees version 1.5.0 or later.

Installing the module

back to top
Upload the folder 'fancy_research_links' to your server's webtrees/modules_v3 folder. Then login to webtrees as administrator and go to Module Administration (Admin -> Module Administration). Look down the list of installed modules for "Fancy Research Links" and ensure it is ticked as "enabled".

After the module is installed go to administration/modules/sidebar to change the position of this tab relative to the other sidebar tabs. Here you also can set the access level. A setting of 'show to editors' or higher will be suitable in most cases.

Updating from a previous version

back to top

In case of updating you can just overwrite your previous version of the Fancy Research Links module with the new one by uploading the folder fancy_research_links' to your server's webtrees/modules_v3 folder. All settings will be remained after an update. It is always a good behaviour to clear the cache after each update. Go to your webtrees administration - clean up data folder and tick the box 'cache'.

Themes

back to top

There is a base stylesheet added to the module which should work in any theme.

Custom translations

back to top

We have included only two new language strings. To translate these strings into your own language copy the file modules_v3/fancy_research_links/language/en-US.php and rename it to your language code used by webtrees (leave the extension as is). To find the right language code look in the folder webtrees/language. Change the second part of each string (the part after the '=>') to a suitable translation in your language and save the file. You can use any texteditor for this purpose.

The Dutch translation file is already added to the module language folder. If you have made your own translation or updated an existing one, please send me a copy.
