Requirements

back to top
This theme requires webtrees version 1.4.2 or later.

Features

back to top
  • The only theme with a THEME OPTIONS MODULE (from version 1.4.2). You can set several theme options from the backend of your webtrees installation.
  • A nice darkgrey layout with orange and yellow accents.
  • Smooth Top Navigation Menu.
  • Nice font for titles.
  • Colorbox support for pdf files.
  • Better dialog boxes. In other themes the dialog boxes will not open in the middle of the page. In this theme all dialog boxes are opening in the center of the browser window.
  • When you put your email link in a custom textarea, just write the link like this (be sure the button 'code' is clicked when you use the texteditor): <a class="contact_link_admin" href="#">send me a message</a>. Now this custom link will use the same messagebox as the default link in the footer. No need to navigate your visitors away from your site.
  • Optional: Flags for language menu in the header with a nice tooltip system (requires the JustBlack Theme Options Module).
  • Optional: Use a compact menu (requires the JustBlack Theme Options Module).

The JustBlack theme has an eye for detail. Every single item on the frontpages of webtrees is in JustBlack theme style. However if you find an item that is not in the JustBlack theme style please let me know at carmen@justcarmen.nl..

Installation

back to top

Just upload the 'justblack' folder in the zip to the folder 'themes' of your webtrees site.

Go to the admin section of your site and select the theme with the themeselector (if themechanges are allowed).

Make JustBlack theme the default theme

back to top
  • If you want to make this theme the default theme for your site go to 'administration'. Click on 'Site configuration' in the menu and set 'default theme' to JustBlack.
  • Click on your family tree in the menu and check the themesettings in the tab 'access'. The theme setting has to be either 'default' or 'JustBlack'. But 'default' is preferable if you want to use the same theme througout your whole site. If you want to change the default theme in the future you only have to change the theme settings in the site configuration.
  • If you have more then one family tree repeat this step for each tree.
  • If you want to let your visitors select there own theme go back to 'Site configuration' and set 'allow users to select their own theme' to 'yes'.If you want the JustBlack theme to be the default theme for everyone with no options to change the theme set 'allow users to select their own theme' to 'no'. In that case there will be no theme selector in the header and no them dropdownbox on the 'my account' page.
  • Note: already registered users may have selected another theme in the past. User settings takes over family tree settings which take over Site settings.You have to check for each user if their theme setting is set to 'default theme'. If not change it to default to prevent the user using another theme in the future.

Upgrade

back to top
  • Backup your website
  • Download the newest version of JustBlack theme from the theme section of http://webtrees.net
  • If you had added custom files to the justblack theme, like a custom header download these files to your pc.
  • DELETE THE OLD JUSTBLACK THEME FROM THE SERVER. You will find the theme folder in the directory 'themes'.
  • Unpack the downloaded zip file and upload the 'justblack' folder in the zip to the directory 'themes' of your webtrees site.
  • Upload your custom theme files back to the justblack theme folder.

Note: DO NOT JUST OVERWRITE THE OLD JUSTBLACK THEME BECAUSE IN THAT CASE DELETED FILES WILL STILL EXIST AFTER UPGRADE.
THESE FILES CAN CAUSE INSTABILITY.

Options

back to top

NEW: THIS THEME COMES WITH A SEPARATE MODULE WHERE YOU CAN SET SEVERAL THEME OPTIONS.
In the zip you will find an extra folder 'justblack_theme_options'.
Read the ReadMe file in this folder for instructions on how to install and use the module.
Note: This theme will work perfectly well without the module. The module is just extra.

There is a psd file of the header available in the zip 'extra' folder. If you are familiar with photoshop you
can modify the header to your needs. For those who are not familiar with photoshop you will find an empty header (with
no text) in the same 'extra' folder. You can add your own text to the image with any graphical programm or leave it as is and use the built-in webtrees title text.
Rename the file to 'header.jpg' and upload it to the folder 'themes/justblack/css/images'. Note the subfolder 'css'.
Of course you can replace the whole header with one of your own. Make your own header.jpg file and upload it to the folder
'themes/justblack/css/images'. The size of the header is 800 x 150px (from version 1.4.2 uploading a custom header image can be done
from the admin in the JustBlack Theme Options section (requires the separate installation of the JustBlack Theme Options Module).
